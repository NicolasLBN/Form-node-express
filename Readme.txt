How start a project: 
- npm init => Create a node project
- npm i --save express => express is a nodejs framework that allow to build node app
- npm i -g nodemon => nodemon allows to update by itself the project whithout needing to restart the server
- npm i --save ejs => template motor api ejs
- npm i --save body-parser => middleware request parser library
- npm i express session => Create cookies 
- npm i moment => date format library
- npm run start => launch the server
